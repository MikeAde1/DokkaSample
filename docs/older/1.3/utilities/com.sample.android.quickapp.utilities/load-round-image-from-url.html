loadRoundImageFromUrl

fun ImageView.loadRoundImageFromUrl(url: String)

An extension function on ImageView receiver to load an image from remote url and crop it in Round shape

This function uses Glide which is a 3rd party library to handle all the networking and caching side of things. It handles the loading w.r.t lifecycle of the view and crops the loaded image in Round shape.

Sample Usage

yourImageView.loadRoundImageFromUrl(url = "https://someurl.com/someImage")
